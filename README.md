## Getting Started

Clone this repo locally with the following command:

```bash
git clone git@github.com:elijahnikov/pension.git
```

1. Install dependencies using npm/pnpm/bun/yarn:

```sh
bun i
```
```sh
npm install
```
```sh
pnpm install
```
```sh
yarn
```

1. Run locally

```sh
bun run dev
```
```sh
npm run dev
```
```sh
pnpm run dev
```
```sh
yarn dev
```
